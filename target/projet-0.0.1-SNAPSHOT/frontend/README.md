
Ce projet consiste à développer le frontend d’une application web  permettant de gérer des personnes (CRUD) en consommant un backend RESTful développé en Java avec **JAX-RS (Jersey).  
L’application permet d’ajouter, modifier, supprimer, afficher et rechercher des personnes stockées dans une base de données MySQL, en respectant l’architecture Client / Serveur.



##  Technologies utilisées

### Backend
- Java JEE
- JAX-RS (Jersey)
- JDBC
- MySQL
- Apache Tomcat
- Maven

### Frontend
- HTML5
- CSS3
- JavaScript
- Fetch API



## Instructions pour exécuter le projet

### 1️⃣ Lancer le backend
1. Importer le projet dans Eclipse (ou autre IDE Java)
2. Configurer la base de données MySQL
3. Démarrer le serveur **Apache Tomcat**
4. Vérifier que l’API REST est accessible via : http://localhost:8080/projet/api/users/affiche 

### 2️⃣ Lancer le frontend
1. Ouvrir le fichier `index.html` dans un navigateur
2. Le frontend communique automatiquement avec le backend REST
3. Tester les fonctionnalités :
   - Affichage des personnes
   - Ajout
   - Modification
   - Suppression
   - Recherche
